# 👑 Showroom Premium | PATRIC SOFT ®

Sistema de catálogo digital e-commerce optimizado para ventas por WhatsApp. Diseñado con una estética de alta gama en **Black & Gold**, enfocado en la conversión y facilidad de uso para el cliente final.

## 🚀 Características Principales

* **Interfaz Premium:** Diseño moderno, oscuro y minimalista con detalles dorados.
* **Filtros Inteligentes:** Categorización de productos (Tecnología, Hogar, Belleza, Accesorios).
* **Carrito de Compras Dinámico:** Gestión de pedidos en tiempo real con barra de resumen.
* **Cálculo de Financiación:** Sistema automático de cuotas con recargo del **15% de ganancia**.
* **Gestión de Señas:** Detalle automático en el mensaje para reserva del 50% o pago de primera cuota.
* **Integración con WhatsApp:** Envío de pedidos detallados directamente al número de ventas.
* **Optimización de Carga:** Código ligero (Vanilla JS) con carga diferida de imágenes para máximo rendimiento.

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica.
* **CSS3:** Estilos personalizados, gradientes radiales y diseño responsive.
* **JavaScript (ES6+):** Lógica funcional del carrito y procesamiento de pedidos.
* **FontAwesome:** Iconografía profesional.
* **Google Fonts:** Tipografía de diseño (Bebas Neue & Poppins).
